# Morse Code Trainer

Static Morse code trainer built for GitHub Pages deployment.

## Features

- Listening mode with optional live current-character display
- Transcription mode with scoring, mistake highlighting, and on-screen keyboard
- Playback history stored in `localStorage`
- Pause/resume, random seed support, JSON/CSV export, dark mode, and mobile layout
- History is saved from characters that fully completed playback only

## Run locally

Open `index.html` in a browser, or serve the folder with any static file server.

## Deploy to GitHub Pages

1. Push the contents of this repository to GitHub.
2. In repository settings, enable GitHub Pages for the branch that contains these files.
3. Use the root directory as the publish source.

No build step is required.
